# pallet-thread

A library to simplify working with Java threads from clojure.

## Usage

[API docs](http://pallet.github.com/pallet-thread)

## Installation

To use pallet-thread, add the following to your :dependencies:

[pallet-thread "0.1.0-SNAPSHOT"]
